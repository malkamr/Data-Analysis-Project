# 📊 Business Data Analysis – Retail/FMCG Case Study


## Overview


This repository contains my graduation project from the **Creativa Data Analysis Workshop**.


The project focuses on applying the data analysis process to a Retail/FMCG business scenario. It demonstrates how raw data can be transformed into meaningful insights through data cleaning, analysis, visualization, and dashboard development.


## Project Contents


* 📑 Business Data Analysis Presentation (Retail/FMCG)
* 🧹 Data Cleaning
* 📈 Interactive Dashboard
* 💡 Business Insights and Recommendations
